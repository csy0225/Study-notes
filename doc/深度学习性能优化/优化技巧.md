# 深度学习性能优化方法技巧
1. 优化一: 与训练代码相结合，往往有的时候因为各种深度学习框架平台的不同，算子支持的种类不同，导出的推理模型可能不同，但是他们本质上都是同一个算法，在数学逻辑上是等价的。这种不同会导致同一个算法映射出不同算子，有的算子可能是碎片化的，不利于推理，这时候可以看是否将这些细粒度的算子融合成一个等效的大算子。
